Installation instructions
To run this game effectively you must be using a machine with at least ;
• 2 GHz Intel Core i7 - Processor
• 4 GB 1600 MHz DDR3 - Memory
• 500 MB - Storage
To run this game you must have a working text editor such as Idle or atom, running
python 3.6. “https://www.python.org/downloads/”
1. Install Pygame “http://www.pygame.org/download.shtml”
2. Download the game folder named “Zombie Rush”
3. Copy to desktop or desired directory
4. Open the “Zombie Rush” folder and open with idle or atom the file named
“Main.py”
5. Once opened run the module
6. You should then see a start screen, press any key to launch the game.
